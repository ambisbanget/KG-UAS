# KG-UAS
(6026242003 - Hani'atul Khoiriyah) (6026242005 - Daris Irfan Putrawanto)  (Mata Kuliah: Topik Knowledge Graph)

Knowledge Graph ini disusun khusus untuk tujuan akademik dan pembelajaran dengan menggunakan data sumber terbuka berupa artikel berita. Seluruh identitas nama individu dalam graf telah dianonimkan dan seluruh label seperti saksi atau diduga semata-mata merepresentasikan redaksi media pada saat publikasi, bukan putusan hukum, dan bukan sebagai tuduhan, serta tidak menyatakan kebenaran faktual. Kode scraping dan pemrosesan otomatis yang digunakan dalam proyek ini disediakan sebagai alat bantu analisis. Oleh karena itu, setiap hasil ekstraksi isu, status, atau keterkaitan yang dihasilkan oleh kode tersebut wajib diverifikasi ulang oleh pengguna sebelum digunakan lebih lanjut. Struktur graf dan keluaran analisis dalam proyek ini tidak dapat dan tidak boleh dijadikan dasar hukum, alat pembuktian, rujukan penilaian hukum, ataupun dasar pengambilan keputusan yang berdampak hukum.
